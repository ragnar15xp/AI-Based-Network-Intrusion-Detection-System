# AI-Based Network Intrusion Detection System

This project is developed as a Major Project using Machine Learning techniques.

## Objective
To detect malicious network traffic and classify it as normal or intrusion using a Random Forest classifier.

## Technologies Used
- Python
- Machine Learning (Random Forest)
- Streamlit
- Scikit-learn
- Pandas, NumPy

## Features
- Interactive GUI
- Model training
- Intrusion detection
- Performance evaluation using accuracy and confusion matrix

## Execution
python -m streamlit run nids_main.py
